## Development

`efg` can be developed with either `cabal` or `nix`.

### Cabal

- build: `cabal build`
- run: `cabal run efg` 
- test: `cabal test`

### Nix
- build: `nix build .`
- run: `nix run .` or `nix run github:japiirainen/efg`
- development environment: `nix develop .` or `nix develop github:japiirainen/efg`
